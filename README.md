# Arogya-Vahini
Rural Health Referral & Digital Health Platform

## Problem
Rural areas often lack access to qualified doctors and proper healthcare systems. Patients face difficulties in booking consultations and getting medical advice.

## Solution
Arogya-Vahini is a digital platform that connects rural patients with doctors through an easy-to-use web application.

## Features

Patient Side
- User Registration & Login
- Book Doctor Appointment
- Online Consultation
- View Medical History
- Digital Health Records

Doctor Side
- Secure Doctor Login
- Manage Appointments
- View Patient Details
- Provide Medical Advice
